# Flexbox Card

A high-performance, responsive glassmorphic card featuring fluid 3D tilt interactions and a slow-motion animated gradient background.

##  Features
* **Glassmorphism:** Uses `backdrop-filter` for a frosted-glass effect.
* **Fluid Motion:** Custom JavaScript tilt logic that responds to mouse and touch input.
* **Animated Background:** A 400% scale CSS linear gradient that shifts smoothly in slow motion.
* **Responsive:** Fully functional on mobile, tablet, and desktop.

##  Built With
* HTML5
* CSS3
* JavaScript
